# BMW Car Showcase

Este projeto é uma aplicação web que apresenta o modelo BMW G20 330i com uma experiência interativa em 3D. A aplicação foi construída utilizando React, Three.js, GSAP e TailwindCSS.

[Clique aqui para ir para o site feito!](https://bmw-330i.vercel.app)

## Visão Geral

A aplicação permite aos usuários visualizar um modelo 3D do BMW G20 330i, com animações e interações que melhoram a experiência do usuário. A navegação é suave e responsiva, garantindo uma experiência agradável em diferentes dispositivos.

## Funcionalidades

- Visualização interativa em 3D do BMW G20 330i.
- Animações baseadas em scroll utilizando GSAP.
- Navegação responsiva com um menu interativo.
- Seções informativas sobre o carro, incluindo velocidade e assistência ao motorista.

## Instalação

Para rodar este projeto localmente, siga os passos abaixo:

1. Clone o repositório:
    ```sh
    git clone https://github.com/yKaayo/bmw-car.git
    cd bmw-car
    ```

2. Instale as dependências:
    ```sh
    npm install
    ```

3. Inicie o servidor de desenvolvimento:
    ```sh
    npm run dev
    ```

4. Abra o navegador e acesse `http://localhost:3000`.

## Uso

- Navegue pelas seções utilizando o menu.
- Role a página para ver as animações interativas do modelo 3D.
- Clique nos botões de chamada para ação para explorar mais funcionalidades.
